# training_data

This folder stores generated RL training artifacts. `redis/` holds lean ghost-league weight exports and metadata, `redis-server/` holds Redis AOF data, `RLlib/` holds Tune/RLlib checkpoints, and `W&B/` holds local observability output.

Most contents are large, generated, and gitignored. Keep this folder for local runs, but do not treat it as source code.
