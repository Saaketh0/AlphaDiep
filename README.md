# AlphaDiep

AlphaDiep is a custom implementation that I made to make an agent able to play the game diep.io (https://diep.io/)

This repo contains the original typescript code, the headless C++ code I made, as well as the training scripts for training an agent.

I used Ray RLlib for the agents and training, using a RPPO and IPPO model+architecture for training.
Feel free to DM for any more info!

Technical details live in [TRAINING.md](./TRAINING.md).

Big thanks to this repo for having the original Typescript server code, which I used as a base for future C++ headless code: https://github.com/abcxff/diepcustom
